Que puis-je faire pour vous ?

Je suis consultant et développeur full-stack spécialiste du Web.

Je fais le tour des métiers du Web depuis 2006 et suis établi en Freelance depuis 2012. Je peux vous accompagner sur toutes les étapes de votre projet : recueil et définition du besoin, architecture technique, développement et intégration, hébergement et maintenance.

Mes technos : principalement PHP, Python, Javascript et bien sûr HTML/CSS. Je dispose également de compétences en développement d'applications mobile iOS, ainsi qu'en électronique / domotique (plateforme Arduino).

Je travaille depuis toujours quasi-exclusivement avec des produits open source et leurs communautés.

Mes points forts : sérieux, fiabilité et capacité de conseil sur un grand nombre de sujets.

 

Développeur full-stack ?

Un développeur full-stack maitrise l'ensemble des domaines techniques constitutifs d'un projet informatique : administration système et hébergement, modèles de données, logique métier, interface utilisateur (avec l'aide d'un graphiste), expérience utilisateur (UX), recueil et compréhension de besoins.

Pour aller plus loin : http://www.laurencegellert.com/2012/08/what-is-a-full-stack-developer/

Technos

Ayant bien bourlingué dans le domaine du Web, j'ai eu la chance de toucher à pas mal de technologies différentes.

Pour votre projet, je saurai m'adapter à vos besoins et le cas échéant à vos contraintes. "The right tool for the job" est quelque chose de très important à mes yeux.

Toutefois, j'ai mes outils favoris :

  • Le micro-framework Flask écrit en Python pour la gestion du MVC ou simplement pour coder une API REST : un langage à la syntaxe simple et claire et un framework qui me fournit juste l'essentiel pour démarrer une application Web en quelques minutes, sans m'imposer ses choix ;
  • La base de données NoSQL mongoDB pour le stockage et le requêtage des données : grosse communauté, simplicité et capacité à supporter de fortes charges ;
  • Le framework CSS Twitter Bootstrap pour le prototyping et l'intégration HTML : composants réutilisables, bonnes pratiques incluses, rapidité de conception et de réalisation ;
  • Le framework Javascript AngularJS : création simplifiée d'interfaces très dynamiques ;
  • Un hébergement Cloud chez Heroku pour un déploiement continu des développements et une architecture scalable.

Mais j'ai bien d'autres cordes à mon arc ! Go, Node.js, Drupal, Symfony, eZ Publish, Code Igniter, WordPress, Jekyll, Less, RethinkDB, MySQL, PostgreSQL, jQuery...

Et si vous n'avez rien compris aux acronymes et autres noms barbares sur cette page, pas de panique : mon métier c'est aussi de vous expliquer et de vous conseiller !
